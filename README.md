# Data Cleaning & Reporting Automation

## 📌 Overview
This project automates:
- Data cleaning
- Handling missing values
- Removing duplicates
- Generating reports

## 🚀 How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the project:
python main.py

## 📊 Output
- cleaned_data.csv
- report.xlsx
- sales_chart.png

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Excel Reporting
